1. Introduction

FinPi ("FinPi", "we", "our", or "the platform") is a financial technology platform built for the Pi ecosystem. FinPi provides digital financial utilities including payment tools, transaction management, financial organization tools, and ecosystem services designed to improve user experience within the Pi ecosystem.

2. Information We Process

To provide FinPi services, we may process the following categories of information:

Account Information

  • Pi username or identifier provided through Pi Authentication
  • Profile information selected by the user
  • Authentication status

Transaction Information

  • Payment requests and transaction references
  • Payment status and confirmation data
  • User activity related to FinPi features

Device Information

  • Device type and browser information
  • Security preferences and application settings

User Preferences

  • Language selection, theme settings, notification preferences

3. Pi Authentication

FinPi uses Pi Authentication when available. FinPi does not request Pi passwords, Pi passphrases, private keys, or store wallet credentials. Users authenticate exclusively through approved Pi ecosystem authentication methods.

4. Data Security

FinPi applies industry-standard security practices including:

  • HTTPS encryption for all communications
  • Secure server-side processing
  • Environment-protected credentials
  • Access control and security monitoring
  • Data minimization principles

5. User Control

Users may review their information, update preferences, request account deletion, and contact support regarding any privacy concerns at any time.

6. Third-Party Services

FinPi may use trusted service providers for hosting, security infrastructure, analytics, and external reference data (such as cryptocurrency price feeds). Third-party services are used only to support core platform functionality.

7. Reference Price Data

FinPi displays Pi price reference data sourced from third-party providers such as CoinGecko. This data is for informational purposes only and does not constitute financial advice or investment guidance.

8. Policy Updates

This Privacy Policy may be updated as FinPi evolves. Continued use of FinPi constitutes acceptance of updated terms.

9. Contact

For privacy concerns or data requests, contact FinPi Support at finpi.app@gmail.com.
