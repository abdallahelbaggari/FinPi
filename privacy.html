Privacy Policy

FinPi respects the privacy of all users within the Pi Network ecosystem.

FinPi only accesses basic information provided by the Pi Network SDK, such as the user's Pi username, for authentication purposes.

FinPi does not collect personal information such as passwords, private keys, or financial credentials.

All Pi payments are securely processed using the official Pi Network SDK.

FinPi does not sell, trade, or share user information with third parties.

This application currently operates in the Pi Network Testnet environment for testing and development.

By using FinPi, you agree to this privacy policy.
